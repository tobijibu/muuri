更新日: 2019.12.19

send

send処理終了時に、元のグリッドに対してトリガーされます。 (grid.send()メソッドが呼び出さた後、 またはアイテムが別のグリッドにドラッグされた時にトリガーされます。) このイベントはアイテムのレイアウト開始前に呼び出されることに注意してください。


パラメータ

data.item : Muuri.Item
移動したアイテム
data.fromGrid : Muuri
アイテムの移動前のグリッド。
data.fromIndex : number
アイテムの移動前のインデックス。
data.toGrid : Muuri
アイテムの移動先のグリッド。
data.toIndex : number
アイテムの移動先のインデックス。

grid.on('send', function (data) {
  console.log(data);
});

デモ

grid1のアイテム1をgrid2の末尾に移動し、 sendイベントを呼び出し、console.logを出力します。

JavaScript
var gridA = new Muuri('.grid1');
var gridB = new Muuri('.grid2');
gridA.on('send', function (data) {
  console.log('send');
});
gridA.send(0, gridB, -1);
