isPositioning()

item.isPositioning()

アイテムが配置中かどうかを返します。


パラメータ

無し

戻り値

boolean
アイテムが配置中かどうかを返します。 trueの場合は配置中、 falseの場合は配置中ではありません。 

var isPositioning = item.isPositioning();

デモ

アイテムをドラッグし、移動アニメーション中に"isPositioning"ボタンを押すと、 console.logを出力します。

JavaScript

var grid = new Muuri('.grid', {
  dragEnabled: true,
  layoutDuration: 3000
});
 
document.querySelector('.is-positioning').addEventListener('click', function(){
  var firstItem = grid.getItems(0)[0];
  if (firstItem.isPositioning()) {
    console.log('item is positioning');
  }
});
