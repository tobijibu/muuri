dragAxis

アイテムをドラッグした時に、 水平方向または、垂直方向への移動に制限します。 'x'を指定した場合、水平移動になり、 'y'を指定した場合、垂直移動になります。 デフォルトでは、水平方向、垂直方向共に移動することができます。

  • 初期値 : null
  • 値 : string
  • 指定可能な値 : 'x'、'y'
// ドラッグした時に水平方向のみの移動します
var grid = new Muuri(elem, {
  dragAxis: 'x'
});
// ドラッグした時に垂直方向のみの移動します
var grid = new Muuri(elem, {
  dragAxis: 'y'
});
