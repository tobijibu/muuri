showEasing

更新日: 2018.09.05

表示アニメーションのイージングを指定します。 イージングはこちらの値を指定することができます。

  • 初期値 : ease
  • 値 : string
var grid = new Muuri(elem, {
  showEasing: 'cubic-bezier(0.215, 0.61, 0.355, 1)'
});

デモ

"アイテム表示/非表示"ボタンを押すと、それぞれのアイテムに指定されたイージングで表示されます。
※少し分かりづらいかもしれません。

ease
ease-in
ease-out
ease-in-out
JavaScript
var grid1 = new Muuri('.grid1', {
  showEasing: 'ease'
});
var grid2 = new Muuri('.grid2', {
  showEasing: 'ease-in'
});
var grid3 = new Muuri('.grid3', {
  showEasing: 'ease-out'
});
var grid4 = new Muuri('.grid4', {
  showEasing: 'ease-in-out'
});

document.querySelector('.show-items').addEventListener('click', function () {
  toggle(grid1);
  toggle(grid2);
  toggle(grid3);
  toggle(grid4);
});
function toggle(grid) {
  grid.getItems().forEach(function (item, i) {
    if (item.isVisible()) {
      grid.hide(item);
    } else {
      grid.show(item);
    }
  });
}
